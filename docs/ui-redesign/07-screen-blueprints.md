# Screen blueprints

- Today Rhythm: compact context header → Rhythm Field → daily interpretation → morphing check-in action → seven-day signal strip.
- Guided Check-in: six focused steps → review → save transformation and immediate store update.
- Cycle Landscape: Rhythm mode timeline by default → edge-to-edge Month mode → anchored contextual detail.
- Signal Stories: conclusion → visual → reason → confidence/source → next action.
- Intelligence Canvas: contextual header → structured response blocks → dynamic prompts → compact composer.
- Personal Rhythm Calibration: every onboarding step uses a different interaction and composition.
- Membership Reveal: three product-value chapters → transparent plan choice → restore/legal access.
- Personal Control Center: full-width grouped rows, inline controls, and explicit dangerous-action hierarchy.
