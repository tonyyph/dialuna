# Color system

Raw colors live in `rawColors.ts`; screens consume only semantic colors through `useTheme()`.

- Canvas: night 950/900 or pearl 50/100.
- Brand: violet 400/500.
- Intelligence: aqua 400/500.
- Biological: coral 400/500.
- Phase signals pair hue with shape, label, or density so color is never the only carrier.
- A screen chooses one dominant color story; supporting colors remain functional.
- Normal text targets WCAG AA contrast; muted copy is never produced only by extreme opacity.
