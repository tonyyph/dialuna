# Art direction

## Personality and mood

Quietly intelligent, intimate, fluid, assured, and editorial. The interface should feel like observing a living biological system in a private night studio.

## Difference

Dialuna combines personal wellness language with spatial signal visualization. It does not decorate screens with moons; lunar rhythm appears through changing arcs, orbital continuity, density, and cadence.

## Visual language

- Dark foundation: deep blue-black night, never warm brown.
- Light foundation: cool pearl, never yellow cream.
- Violet expresses personal rhythm and primary interaction.
- Aqua expresses intelligence and confidence.
- Coral expresses biological signal and attention.
- Canvas and embedded content are preferred over card stacks.
- Icons use one restrained Ionicons outline language; emoji is not structural UI.
- Illustrations are abstract fields, phase arcs, constellations, and signal traces.
- Data uses phase landscapes, ribbons, constellations, and narrative comparisons.

## Composition rules

One primary signal and one primary action per screen. Full-bleed sections establish place; embedded content groups related information; raised/floating surfaces are reserved for context and action. Each main screen must have a distinct silhouette.
