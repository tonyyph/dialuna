# Motion

Motion communicates continuity, physical response, hierarchy, and data transformation. It uses centralized duration and spring tokens and avoids blanket entrance animations.

- Rhythm transition: field density, arc, and focal point transform with date/phase.
- Check-in transformation: answers consolidate into a review signal.
- Navigation continuity: active field moves between destinations.
- Reduced Motion removes ambient movement and replaces spatial transforms with short crossfades.
- Animation stays on transform/opacity and Reanimated's UI thread.
