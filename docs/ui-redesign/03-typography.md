# Typography

Manrope is the primary typeface for headings, body, labels, and numeric data. Cormorant Garamond is limited to an editorial accent role.

The scale follows displayXL 52/56, displayL 42/47, titleXL 32/38, titleL 26/32, titleM 21/27, bodyL 17/25, bodyM 15/22, labelL 14/18, labelM 12/16, and micro 11/14. Body copy is not shrunk to manufacture a premium feeling. Data uses stable, readable figures.
