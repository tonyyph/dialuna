# Component system

Primitives: AmbientCanvas, MotionPressable, AnimatedMetric, ContextLayer.

Patterns: RhythmField, SignalStrip, PhaseRibbon, ContinuumSelector, AdaptiveDock, ContextHeader, ProgressiveSection.

Feature components: CheckInFlow, CycleLandscape, SignalStory, BodySignalPicker, FloatingComposer, InsightPreview.

Components are created for stable interaction or visual contracts, not merely to reduce line count. Variant unions are preferred over boolean-prop matrices.
